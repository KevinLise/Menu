# Sashi & Kent – Restaurant Landing Page

A responsive landing page for **Sashi & Kent**, a fine Japanese cuisine restaurant.

## Description

Single-page frontend built with semantic HTML5 and CSS3. Dark, elegant design with gold tones reflecting the restaurant's premium identity.

## Technologies Used

- HTML5 (semantic tags)
- CSS3 (variables, Flexbox, Grid, media queries, animations)
- JavaScript (hamburger menu)
- Google Fonts: Cormorant Garamond + Lato
- Font Awesome 6 (icons)

## Sections

| Section | Tag used |
|---|---|
| Navigation | `<header>` + `<nav>` |
| Hero | `<section class="hero">` |
| Events & Schedule | `<section class="events">` + `<table>` |
| About Us | `<section class="about">` + `<article>` |
| Gallery | `<section class="multimedia">` + CSS Grid |
| Sponsors | `<ul>` |
| Contact & Social | `<footer>` |

## How to Run Locally

1. Download all files to the same folder
2. Open `index.html` in any modern browser

```bash
# Optional: local server
npx serve .
```
